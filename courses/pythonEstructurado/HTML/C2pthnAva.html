Clases y objetos ¶

En la programacin orientada a objetos (POO) el concepto ms importante es el de clase ya que es con esta con la que se debe partir para construir cualquier objeto. Podemos pensar en una clase como un 'plano de construccin' en el que se define qu y cmo debe funcionar cierto tipo de objetos. Una clase contiene todas las funciones y variables necesarias para que un objeto realice las acciones que necesitamos que haga.

Un objeto es una instancia de la clase, o sea, un elemento que existe dentro de nuestro programa y fue construido usando las especificaciones de la clase. Si an no te queda cien por ciento claro, ms adelante daremos ejemplos de esto.

Dentro de una clase tenemos dos elementos muy importantes, lo atributos y los mtodos. Hasta este momento nos hemos estado refiriendo a muchos mtodos como 'funciones' y es que en el fondo son muy parecidos pero cabe destacar que slo los mtodos estn asociados a los objetos. Por otro lado, los atributos son variables que alojan las caractersticas especficas de un objeto.

Tanto los atributos como los mtodos se 'llaman' usando el nombre del objeto, un punto y el nombre del atributo o el mtodo.

Veamos un ejemplo de cmo crear una clase usando la palabra reservada class.

class NuevaClase:
    def __init__(self, atrib1, atrib2): # Este mtodo se llama constructor y se ejecuta cada que
        self.atrib1 = atrib1            # se crea un objeto de esta clase, aqu definimos nuestros atributos '
        self.atrib2 = atrib2

    def primerMetodo(self):
        print(self.atrib1 + self.atrib2)

Como observaste, el primer 'mtodo' en ser definido es init() tambin llamado constructor ya que se ejecuta siempre que se crea un nuevo objeto de su clase y define los atributos que contendr. Los parmetros que recibe son self y el valor de los atributos. Self es una palabra que hace referencia al mismo objeto que se est creando y no tenemos que indicarla al crear un objeto. El valor de los atributos lo proporcionamos al crear el objeto y se guardarn en los atributos que estn ah definidos. Veamos un ejemplo de cmo crear un objeto de esta NuevaClase.

objeto1 = NuevaClase(5,10) # Creamos el objeto1 del tipo NuevaClase con los atributos numricos 5 y 10

print(objeto1.atrib1) # Imprimimos un atributo
print(objeto1.atrib2) # Imprimimos el otro atributo

objeto1.primerMetodo() # Llamamos al mtodo definido en nuestra clase

Herencia ¶

La herencia es otro de los pilares de la POO ya que con ella podemos crear nuevas clases a partir de otras. La idea detrs de esto es muy intuitiva, si creas una nueva clase que hereda de otra ya existente la nueva clase tendr los atributos y mtodos de la clase ya existente ms los que se definan en ella. Veamos un ejemplo usando la clase Persona que ya hemos definido.

class Estudiante(Persona):
    def __init__(self, nombre, edad, sexo, carrera): #El constructor recive todos los parmetros iniciales
        Persona.__init__(self,nombre, edad, sexo) #Se mandan los parmetros tiles para la clase 'padre'
        self.carrera = carrera # Se crea un nuevo atributo llamado 'carrera' que slo existe en esta nueva clase

    def estudios(self): #Este es un nuevo mtodo nico de la clase Estudiante
        print(self.nombre + ' estudia la carrera en '+ self.carrera)

La manera de utilizar esta clase es la siguiente:

e1 = Estudiante("Bernardo Jimenez", 21, "masculino", "Ciencias sociales") #Se proporcionan los valores de los atributos

print(e1.informacion()) #Mtodo originario de la clase Persona
e1.estudios() #Mtodo original de la clase Estudiante

Por ltimo vamos a ver cmo modificar un atributo de un objeto ya existente. Simplemente debemos utilizar el nombre del atributo a modificar y asignarle otro valor, de la forma:

e1.nombre = "Miguel Savedra"
